End-to-end encrypted file decryption. Everything runs in your browser — nothing is uploaded, nothing is logged.

Use Chrome, Edge, or Firefox 111+. This tool needs the File System Access API to stream large files to disk without crashing the tab. Safari isn't supported — install the age CLI and run age -d -o out file.age instead.

How it works

Client-side only. The file and passphrase never leave your browser. All decryption happens locally in a Web Worker using the audited typage library (age encryption: scrypt + ChaCha20-Poly1305).

Passphrase handling. Whoever sent this file should have shared the passphrase over a separate secure channel (phone call, Signal, in person) — never alongside the file itself. If they sent both in the same place, the encryption bought you nothing.

Large files. Decryption streams straight to disk via the File System Access API, so a 1 GB file uses only a few megabytes of memory. Requires Chrome, Edge, or Firefox 111+.

Paranoid mode. Save the page source to disk, disconnect your network, open the saved copy, and decrypt offline. The tool will work without internet — proof that nothing phones home.
