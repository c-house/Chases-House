UNO

Saul Bass meets a 1978 game-show splash.

The cards stay loud — saturated red, yellow, green, blue. The frame stays warm — felt-table greens with brass-gold bezels off the chases.house token set. Bungee (Google Font, slab-italic) owns the wordmark and card numerals; Fraunces + Bricolage stay on UI chrome. The hook is the contrast: pop-bright cards living inside a dignified, dark, warm room.

Card anatomy · DOM + inline SVG (no images)

The 108-card deck, spec'd in CSS.

Every card is a div.card. The numeral is Bungee, skewed −12°, painted with a duplicated darker copy of its color (the chunky-shadow that gives the lettering weight). The cream oval is rotated −22° — the 70s tilt. Action symbols are inline SVG so they scale crisply and recolor on a single CSS variable. Hover any card to see the lift.

Notes for /feature-dev:feature-dev

Handoff.

  • No images. Every card is DOM. The full deck weighs 0 bytes of asset.
  • Card factory. One renderCard({color, value}) in render.js emits the structure used in this preview. Wild and Wild+4 are color: 'wild' with value: 'wild' | 'wild4'.
  • Dynamic state on the card: data-color, data-value, classes .playable / .dimmed / .selected drive the brass-outline pulse, dim-non-playable mode, and selection halo respectively.
  • Color bar in HUD: a single CSS var --current-color on .hud drives the swatch + the discard pile's accent shadow. Wild → set to chosen quadrant.
  • Reduced motion. Site-wide rule already nukes most. Explicit overrides in this file kill the perpetual-loop ones (burst, pip pulse, attract).
  • Bungee. Single weight, ~30KB gzipped from Google Fonts. Acceptable. Falls back to system slab if font fails — numerals lose chunk but stay legible.
  • Accessibility. Number/symbol corner glyphs already present on every card (colorblind-safe). All cards tabindex="0"; UNO! and menu are real <button>s. Curtain announces next player by name to screen readers.
  • Hot-seat curtain uses aria-live="polite" when activated; tap-to-reveal restores focus into the hand fan.
