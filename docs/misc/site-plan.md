# Site Plan

## Structure

```
chases.house/           → Home page (landing/hub)
chases.house/games/     → Browser games
```

## Home Page

Central landing page with links to each section. Keep it clean and minimal.

## Games

Simple browser-based games built with HTML/CSS/JS. No backend required.

**Planned:**
- Tic Tac Toe
- Checkers
- More TBD

**Future ideas:** Leaderboards (using localStorage or a free service), multiplayer via WebRTC, themed game pages.

## Future Sections

Open to expansion. Potential ideas:
- Music / DJ tools
- Blog
- Projects / portfolio
- Links page

## Tech

- Static HTML/CSS/JS (GitHub Pages)
- No build step required (can add one later if needed)
- Deploy by pushing to `main`
