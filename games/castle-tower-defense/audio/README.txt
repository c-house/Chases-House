Files used at runtime:

  arrow_loose.wav     Archer fires
  cannon_thud.wav     Cannon fires
  magic_zap.wav       Mage fires
  frost_shimmer.wav   Frost fires
  gold_pickup.wav     Coin ping (per kill)
  enemy_death.wav     Enemy dies
  castle_hit.wav      Enemy reaches gate
  wave_start.wav      Wave begins
  wave_clear.wav      Wave clears
  victory.wav         Run won
  defeat.wav          Run lost
  ui_click.wav        Button/menu interactions
  bgm_loop.mp3        Looping background music

The audio module gracefully no-ops on missing files — the game stays
playable in silence if these assets are removed.
